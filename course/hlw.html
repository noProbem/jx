    一、培养目标

    学习市场上最为前沿的新型开发技术，课程内容紧密结合5G时代的需求布局，并根据市场需求随时更新迭代授课内容，项目实践始终贯穿教学始终，培养基础扎实、实践能力强的互联网开发人才。

    大纲适用人群:

    大学生岗前实训、互联网从业人员深造、非技术转岗及所有热爱网络编程开发人员。

    二、教学特色

    师资力量强大、教学体系严谨、贴近市场需求、塑造健全人格。

    三、课程详情

    PS基础 : PS软件的认识与熟练操作，让学生熟练使用PS软件，掌握ps基本操作，熟练切图与测量UI相关元素，能够快速将图纸切片输出。

    WEB前端基础 : html基础，html标签、样式调用、字体、盒模型、float、clearfix、background、选择器及其用法、WEB标准与规范、综合应用，合理组织网站文件，网站发布。

    HTML5+CSS3 : HTML5新特性、表单域增强元素、选择器、Canvas、Video、Audio、文字字体相关样式、Animate 动画、keyframe动画、Media queries、Flexbox、rem/vw、原生响应式/手机M端开发。

    BootStrap : 排版样式、栅格系统、图标、菜单、按钮、表单、导航等组件应用、巨幕、页头、缩略图和警告框、进度条、媒体对象和well组件、列表组、模态框、下拉、滚动监听等、项目实训。

    MUI : UI组件、窗口管理、事件管理、Utils、ajax（上拉加载、下拉刷新）、移动端事件应用（点击、长按、滑动、拖动）、前后台数据交互、移动端web、移动端app项目实训。

    javaScript : 语法规则，JavaScript变量，数据类型，表达式和运算符，条件判断if与switch系列语句、continue、break、return、for、while循环及遍历，函数、对象、数组、json、jsonp、Dom与Bom、Ajax。

    jQuery : Jq简介、Jq选择器、Jq DOM及其遍历、表单与表格、Jq Ajax详解、Jq 常用插件、Jq项目综合实践、网页中的DOM模型框架，详解DOM用来遍历文档、获取文档中指定元素的方法。

    sass/less/grunt : sass/less快速入门、语言特性、sass/less函数、grunt初体验、图片压缩、合并、css、js压缩、合并、less转css，watch监听使用sass/less编写复用性强，便于维护的样式代码。

    微信小程序 : 条件渲染、列表渲染、建立模板、icon组件、text组件、进度条组件、表单组件、导航组件、图片、视频、地图组件、画布组件、登录注册界面、API调用。

    SQL/JAVA/PHP : MySql:安装、管理、库与表的增、删、改、查相关语句及操作。PHP:安装与语法简介、php编程中的变量、常量、数据类型、运算符与表达式、选择与循环、json字符串生成、前端如何与后台对接数据等。

    ES6 : let const块级作用用、解构赋值、字符串模板、函数默认参数，扩展运算符、剩余参数、箭头函数、数组循环、 对象的简洁语法、 Promise、Set、Weakset、Symbol、类Class的声明与继承。

    vueJS : Vue简介、Mvvm双向绑定、模板语法、生命周期、条件判断、样式绑定、事件、等指令、全局与局部组件、组件传值与调用、单文件组件、Router状态管理、Vue动画、 Slot插槽、axios、项目实训。

    webpack/npm : nodeJS安装、配置安装文件、js样式与图片的引用、搭建VUE 开发环境，脚手架工具的使用、创建项目npm instal系列命令。
